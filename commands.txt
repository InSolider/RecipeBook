1. Активація віртуального середовища:

.venv\Scripts\activate.bat


2. Запуск серверу Джанго:

py manage.py runserver


3. Створення міграцій (змін) у БД:

py manage.py makemigrations


4. Впровадження міграцій (змін) у БД:

py manage.py migrate


5. Деактивація віртуального середовища:

deactivate


6. Зберігаємо список встановлених пакетів:

pip freeze > env.txt


7. Відмічаємо всі файли для їх подальшого додавання до коміту:

git add .


8. Створюємо новий комміт:

git commit -m ""


9. Пушимо зміни до репозиторію:

git push --all


Додатково:

a. Команда для ствоерння адміністратора:

py manage.py createsuperuser


b. Відкат всіх незакомічених змін до стану останнього коміту:

git reset --hard


b. Видалення всіх невідслідковуваних файлів:

git clean -f
