1. Активація віртуального середовища:

.venv\Scripts\activate.bat


2. Запуск серверу Джанго:

py manage.py runserver


3. Створення міграцій (змін) у БД:

py manage.py makemigrations


4. Впровадження міграцій (змін) у БД:

py manage.py migrate


5. Деактивація віртуального середовища:

deactivate


6. Після змін відмічаємо всі файли для їх подальшої відправки до репозиторію:

git add .


4. Створюємо новий комміт:

git commit -m ""


5. Пушимо зміни до репозиторію:

git push --all


Додатково:

a. Відкат всіх незакомічених змін до стану останнього коміту:

git reset --hard


b. Видалення всіх невідслідковуваних файлів:

git clean -f
