1. Активація віртуального середовища:

.venv\Scripts\activate.bat


2. Деактивація віртуального середовища:

deactivate


3. Після змін відмічаємо всі файли для їх подальшої відправки до репозиторію:

git add .


4. Створюємо новий комміт:

git commit -m ""


5. Пушимо зміни до репозиторію:

git push --all
